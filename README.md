# Dr. Plutte & Mettlach-Plutte – Rechtsanwälte Leverkusen

Улучшенная статическая версия сайта [erbrecht-leverkusen.de](https://www.erbrecht-leverkusen.de), собранная Джарвисом.

## Что изменено и улучшено

- **Современный дизайн**: тёмно-синяя палитра с золотыми акцентами — классика для юридической фирмы, внушающая доверие.
- **Hero-секция**: полноэкранный баннер с чётким заголовком, описанием специализации и кнопкой CTA «Beratungstermin vereinbaren».
- **Карточки по правовым областям**: Erbrecht, Arbeitsrecht, Zivilrecht оформлены в отдельные карточки с иконками — сразу видна экспертиза.
- **Блок философии**: ценности кансцелярии представлены в виде аккуратных блоков с иконками вместо сплошного текста.
- **Контакт**: все контактные данные собраны в структурированную сетку — адрес, телефоны с пояснениями, e-mail, WhatsApp, часы работы.
- **Убраны устаревшие элементы**: упоминание Corona-Pandemie, блок «Technische Störung», блок с cookie-заглушкой.
- **Два телефонных номера** теперь чётко подписаны: кансцелярский и мобильный/аварийный.
- **Mobile-first адаптация**: гамбургер-меню, адаптивные сетки, читаемая типографика на любом экране.
- **Семантический HTML5**: правильные теги `<header>`, `<nav>`, `<section>`, `<address>`, `<footer>`, корректные мета-теги.
- **Без внешних зависимостей**: только HTML + CSS + минимум ванильного JS. Работает на GitHub Pages как есть.

## Структура файлов

```
index.html   — основная страница
styles.css   — все стили
README.md    — описание проекта
```

## Запуск

Откройте `index.html` в браузере или разместите файлы в репозитории GitHub Pages.
